# homebrew-brew-automator

Homebrew tap for [brew-automator](https://github.com/xtruhlar/brew-automator).

## Install

```
brew tap xtruhlar/brew-automator
brew install brew-automator
```

## Usage

```
brew-automator init
brew-automator run
```

See the [main repo](https://github.com/xtruhlar/brew-automator) for full documentation.
